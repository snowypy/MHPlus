# MHPLus

This is made to integrate with the Minehut API which is the Api for Minehut a server hosting & minecraft community service. The api allows you to fetch info simply by providing a server name.

Minehut:
  - discord.gg/minehut
  - minehut.com
  - api.minehut.com
